# Ramble BOT - Traduzioni
Benvenuto nel repository GitHub del bot con le traduzioni!                                                  
Vuoi aiutare il nostro bot di traduzione? Siete nel posto giusto!                                                  
Il bot supporta lingue come:                                                  
- Inglese
- Russo
- Ucraino
- Tedesco
- Polacco
- Francese
- Italiano
- Cinese (semplificato)
- Giapponese
